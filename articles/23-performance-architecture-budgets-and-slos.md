# Performance Architecture: Budgets and SLOs

Performance becomes manageable when tied to explicit budgets.

## Suggested Budgets

- API p95 latency per route class
- max payload sizes for list endpoints
- page-level frontend TTFB and interaction targets

## SLO Examples

- 99.9% auth verification availability
- 99.5% dashboard read availability
- strict error budget for write APIs

## Operational Payoff

Architecture decisions can be evaluated against objective targets.
